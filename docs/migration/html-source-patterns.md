# HTML Source Patterns

This document captures recurring patterns from the original HTML portfolio that should remain recognizable after migration.

## Hero

- two-column desktop layout
- text block on the left, image on the right
- large typing title
- compact social actions under the body copy
- subtle halo/glow behind the image

## Skills

- headline + short intro text
- uniform cards in a tidy grid
- icon on the left, text on the right
- secondary chips row below the main cards

## Timeline

- strong section title
- vertical guide line
- circular icon markers
- date label, heading, body, chips

## Projects

- dark elevated cards
- media flush at the top
- consistent body spacing
- chips above action buttons
- cards should feel aligned even when descriptions differ

## Game And Contact

- game opens as a true modal overlay
- contact area should remain centered, clean, and readable
